Politica sulla Privacy

Ultimo aggiornamento: 4 agosto 2025
Informazioni Importanti: Questa politica sulla privacy descrive come Amazon Scraper Bot raccoglie, utilizza e protegge le tue informazioni personali durante l'utilizzo del servizio di monitoraggio prezzi Amazon.

1. Chi Siamo

Amazon Scraper Bot è¨ un servizio automatizzato che monitora i prezzi dei prodotti Amazon e invia notifiche tramite WhatsApp, Discord e Telegram. Il servizio è¨ fornito per scopi informativi e di risparmio per gli utenti.

2. Informazioni che Raccogliamo

2.1 Dati Raccolti Automaticamente

  • Identificatori Prodotto: ASIN (Amazon Standard Identification Number) dei prodotti che monitoriamo
  • Informazioni Prodotto: Titoli, prezzi, immagini, descrizioni, valutazioni e recensioni dei prodotti Amazon
  • Dati di Mercato: Variazioni di prezzo, disponibilità , offerte lampo, status Prime
  • Metadati Tecnici: Timestamp di scansione, categorie prodotto, domini Amazon (.it, .com, etc.)

3.1 Funzionalitè  Principali

  • Monitoraggio Prezzi: Tracciamento continuo dei prezzi Amazon per rilevare variazioni e offerte
  • Notifiche Personalizzate: Invio di avvisi quando i prezzi scendono sotto le soglie impostate
  • Gestione Database: Archiviazione dello storico prezzi per analisi e trend
  • Ottimizzazione Servizio: Miglioramento dell'accuratezza e velocitè  del monitoraggio

3.2 Elaborazione Dati

  • Analisi dei trend di prezzo e identificazione di opportunità  di risparmio
  • Filtraggio automatico di prodotti non pertinenti o con prezzi anomali
  • Generazione di report e statistiche sui risparmi generati
  • Esportazione dati in formato Excel per analisi avanzate

4. Archiviazione e Sicurezza dei Dati

4.1 Database Locale

Tutti i dati vengono archiviati in un database SQLite locale (products.db) con le seguenti caratteristiche:

  • Crittografia a livello di file system
  • Backup automatici per prevenire perdita dati
  • Controllo degli accessi limitato al sistema di monitoraggio
  • Pulizia periodica di dati obsoleti

4.2 Sicurezza delle Comunicazioni

  • Telegram: Comunicazioni crittografate end-to-end tramite API ufficiali
  • Discord: Utilizzo di webhook sicuri e token bot autorizzati
  • WhatsApp: Automazione tramite WhatsApp Web con autenticazione sicura

5. Condivisione dei Dati

5.1 Non Condivisione

I tuoi dati personali NON vengono mai salvati, visualizzati o condivisi con terze parti, inclusi:

  • Amazon o altre piattaforme e-commerce
  • Servizi di marketing o pubblicitari
  • Broker di dati o aggregatori
  • Altri utenti del servizio

5.2 Accesso Limitato

L'accesso ai dati dei prodotti è¨ limitato esclusivamente a:

  • Sistemi automatizzati per il monitoraggio prezzi
  • Funzioni di notifica per l'invio di aggiornamenti
  • Processi di backup e manutenzione del sistema

6. Conservazione dei Dati

6.1 Durata di Conservazione

  • Dati Prodotto: Conservati fino a quando il prodotto è¨ monitorato attivamente
  • Storico Prezzi: Mantenuto per un massimo di 365 giorni per analisi trend
  • Log Sistema: Conservati per 30 giorni per scopi di debug e miglioramento

6.2 Cancellazione Automatica

Il sistema effettua automaticamente:

  • Pulizia di prodotti non più disponibili dopo 90 giorni
  • Archiviazione compressa per dati storici oltre i 12 mesi

7. Tecnologie Utilizzate

7.1 Scraping e Monitoraggio

  • Web Scraping: Utilizzo di BeautifulSoup e Selenium per raccolta dati pubblici
  • Rate Limiting: Rispetto dei limiti di richiesta per non sovraccaricare i server

7.2 Automazione Messaggistica

  • API Ufficiali: Utilizzo esclusivo di API autorizzate per Telegram e Discord
  • WhatsApp Web: Automazione tramite interfaccia web ufficiale
  • Gestione Errori: Sistemi di retry e fallback per garantire la consegna

8. Modifiche alla Politica

Questa politica sulla privacy può essere aggiornata periodicamente per riflettere:

  • Cambiamenti nelle funzionalità del servizio
  • Aggiornamenti normativi
  • Miglioramenti nelle pratiche di sicurezza
  • Feedback degli utenti

Gli utenti verranno notificati di modifiche significative tramite i canali di comunicazione utilizzati.
